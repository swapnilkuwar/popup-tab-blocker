# Privacy Policy – Popup Tab Blocker

**Last updated:** 2026-02-03

Popup Tab Blocker respects your privacy and is designed to work entirely on your device.

---

## Data Collection

Popup Tab Blocker does **not** collect, store, or transmit any personal or sensitive user data.

---

## What the Extension Does

The extension:

- Detects when a website opens a new tab or window
- Blocks that tab or window **only if the user has enabled blocking for that site**
- Saves user preferences (enabled/disabled sites and block counters) **locally** in the browser

---

## Data Storage

- All settings are stored using Chrome’s local storage API
- No data is sent to external servers
- No cloud storage is used
- All data remains on the user’s device

---

## What We Do Not Do

Popup Tab Blocker does **not**:

- Track users
- Collect analytics
- Show advertisements
- Execute remote code
- Share data with third parties
- Modify website content

---

## Permissions Explanation

Popup Tab Blocker uses the following Chrome permissions:

- **tabs**  
  Required to detect and close tabs or windows opened by websites

- **storage**  
  Required to store per-site user preferences locally

These permissions are used **only** for the extension’s stated functionality.

---

## Open Source

Popup Tab Blocker is open source.  
The source code is publicly available on GitHub and can be reviewed by anyone.

---

## Changes to This Policy

If the extension’s functionality changes in a way that affects user privacy, this policy will be updated accordingly.
